## Week 3 실습 내용

### 1. 타입 변환 실습
- **TypeCastingExample.java**: byte → short → long → float → double 자동 변환
- **WideningCastingExample.java**: byte → short → int → long → float → double 완전 자동 변환
- **ForceCastingExample.java**: double → int (소수점 손실), int → byte (오버플로우)

### 2. 비트 연산 실습
- **BitwiseOperationsExample.java**: AND(&), OR(|), XOR(^), NOT(~) 연산
- **ShiftOperationsExample.java**: 왼쪽 시프트(<<), 오른쪽 시프트(>>), 부호 없는 오른쪽 시프트(>>>)

### 3. 연산자 실습
- **OperatorsExample.java**: 전위/후위 증감 연산자, 삼항 연산자

### 4. 입력 처리 실습
- **ScannerInputExample.java**: Scanner 클래스를 사용한 사용자 입력 (이름, 나이, 키)

### 5. 데이터 타입 및 상수 실습
- **PrimitiveDataTypes.java**: Java 8가지 기본 데이터 타입과 메모리 크기
- **WrapperConstantsExample.java**: Integer, Long, Double의 MAX_VALUE, MIN_VALUE
- **ByteOverflowExample.java**: byte 타입 오버플로우 현상 (127 + 1 = -128)

## 학습 목표
- Java 기본 문법 및 데이터 타입 이해
- 연산자의 종류와 사용법 습득
- 타입 변환의 원리와 주의사항 파악
- 사용자 입력 처리 방법 학습
